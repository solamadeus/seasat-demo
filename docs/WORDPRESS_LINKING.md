
# Linking from WordPress

The easiest route is just a button or menu item that opens the GitHub Pages demo.

## Option A — menu link
- log into WordPress admin
- go to `Appearance` -> `Menus`
- add a `Custom Link`
- paste your GitHub Pages URL, for example:
  `https://solamadeus.github.io/seasat-demo/`
- set the label to `Demo`
- save menu

## Option B — button on a page
Add a button block on your page and link it to the GitHub Pages URL.

## Option C — iframe embed
Only do this later.
For tonight, opening the demo in a new tab is simpler and more reliable.
